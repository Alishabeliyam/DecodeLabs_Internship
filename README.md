# AI Chatbot using Python

## Overview
This project is a simple AI Chatbot developed using Python. The chatbot interacts with users through predefined responses and simulates human-like conversations.

## Features
- Interactive conversation
- User-friendly interface
- Predefined intelligent responses
- Easy to customize and expand

## Technologies Used
- Python
- NLTK (Optional)
- Basic NLP Concepts

## Project Structure
AI-Chatbot/
│
├── chatbot.py
├── responses.py
├── README.md

## Installation

1. Clone the repository
2. Install dependencies

pip install nltk

3. Run the project

python chatbot.py

## Learning Outcomes
- Natural Language Processing basics
- User input handling
- Conditional logic
- Chatbot development fundamentals

- Voice support
